Tentang Saya

Saya adalah seorang Machine Learning Engineer & IT Infrastructure Specialist yang mendedikasikan karir saya untuk membangun solusi komputasi berkinerja tinggi. Fokus utama saya terletak pada konvergensi antara kecerdasan buatan dengan infrastruktur TI berskala besar.

Dengan latar belakang yang kuat dalam optimasi tingkat rendah menggunakan Cython dan manajemen memori (seperti pemrosesan multi-threaded yang melangkahi Global Interpreter Lock), saya memastikan bahwa model AI dan pipeline data berjalan dengan efisiensi maksimal pada perangkat keras bare-metal maupun cloud.

Di sisi infrastruktur, saya gemar mendesain topologi jaringan cloud yang kokoh, mengamankan gerbang data dengan Cloudflare, serta mengefisiensikan sinkronisasi penyimpanan cloud menggunakan tool canggih seperti rclone.
